DirectTrans

Smart clipboard translation & hotkey utility for Windows

1. General Overview

DirectTrans is a system-wide translation utility for Windows. It allows you to select and translate text in any application using global hotkeys.

The application supports four translation providers: Google Free (no key required), Gemini, GroqCloud, and Mistral. The latter three require an API key.

2. Retrieving API Keys

To use AI models, register for an API Key from your preferred provider:

  • Google Gemini: aistudio.google.com/app/apikey
  • GroqCloud: console.groq.com/keys
  • Mistral AI: console.mistral.ai/api-keys

3. Configuring Your API Key

1 Paste the Key

Open the DirectTrans Settings window and select your preferred Provider. Paste your key into the API key input field.

2 Load Available Models

Click the "↻ Load Models" button to filter and list available models for translation.

3 Test & Save Key

Click the "Test" button to verify the connection. If successful, the adjacent badge will turn green and show "Pass". If it fails, it will turn red showing "Fail".

Click "Save Key" to save your configuration. (Note: Successfully Loading Models or Testing will also automatically save your key).

4. Hotkeys & Text Replacement

You can configure hotkeys to translate text with 2 visual modes:

A Popup Mode (Show results)

Select text and press your popup hotkey (e.g. Ctrl+Shift+V). A popup will appear. Click "Copy" to copy the content or "Replace" to overwrite the selected text in your active document.

B Replace Mode (Direct replacement)

Select text and press your replace hotkey (e.g. Ctrl+Shift+E). The selected text will be directly overwritten with its translation instantly without showing a popup window.

5. UI Language Support

The top-right corner of the Settings window features a language dropdown supporting 5 interface languages: Vietnamese, English, Korean, Chinese, and Japanese. The application dynamically updates labels and fonts when changed.
